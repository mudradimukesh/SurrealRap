# SurrealRap

SurrealRap is a Flutter starter app for capturing dreamlike rap ideas, hooks, and performance moods.

## Getting Started

```sh
flutter pub get
flutter run
```

If this project was created from Codex while the local Xcode license was pending, run this once in Terminal before using Flutter tooling on macOS:

```sh
sudo xcodebuild -license
```

Then accept the license and rerun the Flutter commands above.
